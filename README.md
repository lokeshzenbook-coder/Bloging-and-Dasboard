View your app in AI Studio: https://ai.studio/apps/afb183d5-52b3-4d57-b174-f0c4724c21d1

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
